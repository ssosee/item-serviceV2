# 목표
이전 서비스에서 `타임리프`를 사용해서 폼에서 `체크박스`, `라디오 버튼`, `셀렉트 박스`를 편리하게 사용하는 방법을 학습

## 요구사항 추가
* 판매 여부
  * 체크 박스로 선택
    * 판매 오픈 여부
* 등록 지역
  * 체크 박스로 다중 선택
    * 서울, 부산, 제주
* 상품 종류
  * 라디오 버튼으로 하나만 선택
    * 도서, 식품, 기타
* 배송 방식
  * 셀렉트 박스로 하나만 선택
    * 빠른 배송
    * 일반 배송
    * 느린 배송
